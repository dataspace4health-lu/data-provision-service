# Security Policy

We take the security of Data Provision Service seriously. If you believe you have found a security vulnerability, please follow the steps below.

## Reporting a Vulnerability
- Email the security team at seyedziaeddin.alborzi[AT]global.ntt.
- Provide a detailed description of the issue, including reproduction steps, potential impact, and any suggested mitigations.
- Do not create public GitHub issues for security reports.

## Response Process
- We will acknowledge receipt of your report within 5 business days.
- We will investigate and provide status updates as the issue is triaged and resolved.
- Once a fix is available, we will coordinate a disclosure timeline with you.

## Supported Versions
Security fixes are applied to the `main` branch. Please ensure you are using the latest release.
